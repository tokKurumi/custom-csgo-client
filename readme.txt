Для установки конфига потребуется перенести и заменить все все файлы в стандартной папке CSGO вашего компьютера.
Для этого просто перетащите папку csgo в вашу папку игры.
----------------------------------------------------------------------------------------------------------------
Вак бан за это дать не могут, можно использовать и не бояться играть на официальных серверах.
----------------------------------------------------------------------------------------------------------------
При проверке целостности игры, такие аспекты, как радары, радио-меню могут замениться игрой на стандартные. При
Данном действии рекомендуется заного произвести установку данного пака.
----------------------------------------------------------------------------------------------------------------
Если вы используете данный пак в публичных интересах, укажите его автора https://steamcommunity.com/id/KurumiZE/
----------------------------------------------------------------------------------------------------------------
Пак содержит: кучу настроек для вашей игры (рекомендую самим ознакомиться с ними: \custom_csgo\csgo\cfg),
кастомный чат игры, кастомный шрифт (перед использованием придётся установить на свою систему
Windows: \custom_csgo\csgo\panorama\fonts\NunitoSans-Light.ttf), 2 кастомных фона главного меню (cobblestone и
vertigo), кастомные улучшенные радары для игры в соревновательных картах, измененноё радио-меню преимущественно
для игры в режим Zombie Escape на SourceMod'е.
----------------------------------------------------------------------------------------------------------------
Параметры запуска:
+exec ze.cfg -language textmodorel -novid -nojoy -high -console -freq 144 -threads 12
----------------------------------------------------------------------------------------------------------------
+exec ze.cfg -> автоматически запускает конфиг при старте CS:GO
----------------------------------------------------------------------------------------------------------------
-language textmodore -> автоматически устанавливает улучшенный чат для игры (только английская версия),
русская версия скачивается по ссылке https://drive.google.com/file/d/1FoHsL0P4Os6uX8FX4JEYiElMDXbfO0il/view
----------------------------------------------------------------------------------------------------------------
-novid -> автоматически скрывает застаку CS:GO
----------------------------------------------------------------------------------------------------------------
-nojoy -> отключает поддержку джойстиков и стиков для повышения FPS
----------------------------------------------------------------------------------------------------------------
-high -> запуск CS:GO в режима высокого приоритета
----------------------------------------------------------------------------------------------------------------
-console -> запуск консоли при старте CS:GO
----------------------------------------------------------------------------------------------------------------
-freq 144 -> частота обновления кадров монитора
----------------------------------------------------------------------------------------------------------------
-threads 12 -> количество потоков процессора выделенных под запуск CS:GO (не ставить больше 16)
